# Sorting Algorithms

This folder contains classic sorting algorithms with Python implementations + optional visualizations.

### Implemented

- ✅ Bubble Sort
- ✅ Selection Sort
- ✅ Insertion Sort
- ✅ Merge Sort
- ✅ Quick Sort
- ✅ Heap Sort

### How to Run

```bash
python sorting/quick_sort.py
```

### Visualization

```bash
python sorting/visualize_sorting.py
```
