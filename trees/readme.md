# Trees

Tree-based data structures & traversals.

### Implemented

- ✅ Binary Tree (with DFS traversals)  
- ✅ Binary Search Tree (insert, search, delete)  
- ✅ AVL Tree (self-balancing)  
- ✅ Heaps (Min/Max)

### Run Example

```bash
python trees/bst.py
```

### Visualization

```bash
python trees/visualize_trees.py
```
